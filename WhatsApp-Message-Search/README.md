## Problem
Users often need to find old messages (addresses, bills, notes) but chat-based search is inefficient and chaotic.

## Work done:
- User research
- Problem breakdown
- Solution ideation
- Metrics definition
- MVP
- Wireframing

## Target Users
Heavy WhatsApp users with long personal and group chats.

## Research
- 8 user interviews
- Survey (47 responses)
- Secondary research (forums)

## Key Insights
- Users remember context, not keywords
- Search fails for images, forwarded content, and mixed languages

## Proposed Solution
- Smart message clustering
- Context-aware search
- Saved messages & auto-labels

## Success Metrics
- Search success rate
- Time to find message
- Feature adoption

## MVP
- Mid-Fi Wireframing using Balsamiq
- Initial prototype (MVP) using Lovable AI.
